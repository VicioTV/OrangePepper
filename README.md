# Orange Pepper — Landing

Sitio web de **Orange Pepper**, servicio de barra móvil y coctelería de autor para eventos privados en Córdoba, Argentina.

Página única, estática, sin build (HTML/CSS/JS en un solo `index.html`). Usa GSAP, Swiper y Lottie desde CDN.

## Cómo verlo localmente
Abrí `index.html` en el navegador, o serví la carpeta con cualquier servidor estático:

```bash
npx http-server -p 8080
```

## Publicar en GitHub Pages
1. Subí el contenido de esta carpeta a un repositorio.
2. En el repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch**, rama `main`, carpeta `/ (root)`.
3. El archivo `.nojekyll` ya está incluido para que GitHub Pages sirva todos los assets tal cual.
